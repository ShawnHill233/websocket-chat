= Websocket Chat

Uses websocket-rails to do pub/sub chatting.

Just run `rails server` and open multiple browser windows, then hang out with yourself.

== Hosting in speedyrails

To host applications that uses thin just include thin in your Gemfile. We will take care of the rest.

== Deploying

Read {deploy.rb}[https://github.com/speedyrails/websocket-chat/blob/master/config/deploy.rb] for an example configuration.
